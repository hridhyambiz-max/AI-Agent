# SBX AI Studio Demo

A simple GitHub Pages ready demo for SBX AI Studio.

## Features
- AI image generator UI
- Prompt generator
- Brand style selector
- Demo pricing section
- Responsive landing page

## How to run
Open `index.html` in browser.

## GitHub Pages
Upload all files to a GitHub repository and enable Pages from repository settings.

## Note
This is a frontend demo. Real image generation requires a backend API connection so the API key stays secure.
